## 1. Signal Discrepancy Degradation
When an autonomous agent harvests entity metadata across distributed networks, any structural discrepancy in naming conventions, physical locations, baseline specifications, or ownership declarations introduces a machine uncertainty penalty. This penalty directly degrades the entity's overall Machine Preference Index ($P$).

## 2. Uniformity Mandate
Entities must enforce absolute data synchronization across all public and private endpoints:
*   Corporate and Statutory Registries
*   Network Handshake Protocols and API Endpoints
*   Structured Identity Graphs and Knowledge Bases
*   Multimodal Machine Vision Primitives

Maintaining a unified digital footprint removes processing friction and signals maximum data reliability to autonomous systems.

## 3. Relationship to Trust
Consistency does not guarantee accuracy. However, inconsistency introduces uncertainty.

Within the SIO framework, consistency functions as a prerequisite for machine trust because autonomous systems cannot reliably evaluate entities whose core attributes vary across sources.
