## Problem Statement  

- Search engines rank.
- Answer engines answer.
- Generative engines synthesise.
- Visual engines interpret.

Autonomous systems must choose.  

When multiple entities satisfy a given objective, a synthetic intelligence system must determine which entity should be selected for action.  

This selection process introduces a new optimisation challenge not directly addressed by SEO, AEO, GEO, or VEO.  

The Machine Preference Problem is defined as:  

> The challenge of increasing the likelihood that an autonomous system will select a given entity when multiple valid alternatives exist.  

Synthetic Intelligence Optimisation (SIO) is proposed as a framework for addressing this problem.
