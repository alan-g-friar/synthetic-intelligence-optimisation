## 1. Scope
This module defines the architectural and functional parameters of a Synthetic Intelligence (SI) system as evaluated within the SIO framework.

## 2. Functional Agent Stack Boundary
A system qualifies as a Synthetic Intelligence system when its execution architecture operates across three integrated layers:
*   **The Perception Layer:** Continuous ingestion of multimodal signals, including unstructured text, structured metadata, and machine-vision visual primitives.
*   **The Cognitive Layer:** Recursive planning, contextual evaluation, memory retrieval, tool discovery, and multi-agent task orchestration.
*   **The Action Layer:** Programmatic execution via API orchestration, database transactions, and deterministic tool-calling.

## 3. Operational Distinction
Generative systems primarily retrieve, synthesise, or present information. Synthetic Intelligence systems extend beyond information processing by incorporating planning, memory, tool execution, and autonomous action capabilities.

For the purposes of the SIO framework, the defining characteristic of a Synthetic Intelligence system is not content generation but the ability to evaluate alternatives and initiate actions within a defined workflow.

SIO targets this machine preference and decision boundary.
